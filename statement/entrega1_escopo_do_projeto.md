Passo 1 - Veja como instalar o EA (Enterprise Architect) no link "LEIA - Orientações para o projeto" (no Moodle).
Passo 2 - Instale o EA
Passo 3 - Baixe o template no Moodle "Template Enterprise Architect" e abra no EA
Passo 4 - No template você encontrará a estrutura do projeto com todos os elementos estudados. Neste momento deverá ser entregue o projeto no EA (arquivo .eap) com os seguintes elementos:

- Introdução
- Descrição geral do produto
- Requisitos
- Matriz de rastreabilidade RN x RF
